# 🗺️ Notoid Project Roadmap

> **Notoid** is an AI-powered study assistant built with C# and .NET, designed to summarize documents and generate study notes using the **Google Gemini API**.  
> This roadmap outlines the key development milestones, ongoing improvements, and upcoming goals for the project.

---

## ✅ Completed Milestones

- [x] Core desktop app built using **Windows Forms**  
- [x] Integrated **Google Gemini API** for AI-powered summarization  
- [x] Implemented **PDF, DOCX, and PPTX** file parsing  
- [x] Added **PDF generation** using QuestPDF  
- [x] Improved UI and icon handling via embedded resources  
- [x] Comment cleanup and full English documentation (`v1.3.5`)  

---

## ⚙️ In Progress

- [ ] UI/UX polish with **DevExpress components**  
- [ ] Better exception handling and error reporting  
- [ ] Add “Open Recent” and “Clear All” file management options  
- [ ] Performance optimization for large files  

---

## 🚀 Planned Features

### **v1.4 – AI Enhancement & PDF Customization**
- [ ] Improve AI output with better formatting (headings, lists, key points)  
- [ ] Add customizable **PDF templates** (fonts, page style, dark mode)  
- [ ] Refine AI prompt logic for more accurate summaries  

### **v1.5 – User Experience & Data Handling**
- [ ] Add **history panel** to view previous summaries  
- [ ] Save summaries to local database or JSON  
- [ ] Add file drag & drop support  

### **v2.0 – Cloud Integration & Cross-Platform**
- [ ] Integrate **Google Drive** or **Dropbox** sync  
- [ ] Export/import settings and session data  
- [ ] Port to **.NET MAUI** for cross-platform (Windows/macOS/Linux)  
- [ ] Implement built-in **note editor** with AI support

## 💬 Contribution Guide

Contributions, feature ideas, and feedback are always welcome!  
If you’d like to contribute:

1. Fork the repository  
2. Create a feature branch (`feature/new-feature-name`)  
3. Submit a Pull Request  

➡️ You can also open feature requests or report bugs under **[Issues](https://github.com/MetehanSarica/Notoid/issues)**.

---

## 🧭 Roadmap Philosophy

Notoid aims to combine:
- Simplicity of a desktop app 🖥️  
- Power of AI summarization 🤖  
- Flexibility of custom document output 📄  

Future updates will continue to focus on usability, clarity, and developer transparency.
